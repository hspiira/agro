# Alm Agros

Responsive Next.js site for Alm Agros, based on the desktop reference in `references/Mubende Agro Farms.html`.

## Development

```bash
pnpm dev
```

The app runs on the first available Next.js port. In this workspace, an existing server may already be using `http://localhost:3001`.

## Verification

```bash
pnpm lint
pnpm build
```
